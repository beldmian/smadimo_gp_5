# ГП-5. Команда 38

## Датасет

В качестве датасета были собраны данные об объявлениях о сдаче в аренду квартир в г. Москва на сайте [ЦИАН](https://www.cian.ru/).
Для удобства дальнейшей работы датасет был размещен на kaggle, более подробное описание можно посмотреть там же: [Moscow Rent Cian With Images](https://www.kaggle.com/datasets/beldmian/moscow-rent-cian-with-images)

## Бизнес задачи

В рамках проекта бизнес-заказчиком выступает некоторое крайне крупное риелторское агенство ООО "бнал", которое столкнулось с недостаточной оптимизацией собственных процессов.

В рамках проекта было поставлено 2 бизнес задачи в рамках одного общего направления — оптимизация работ по приему объекта для дальнейшей сдачи, а именно:
1. Оптимизировать процесс определения класса ремонта объекта по фотографии (хороший ремонт или "Бабушкин") для более корректного позиционирования и распределения между компетентными специалистами в каждом из классов.
2. Оптимизировать процесс выяснения первоначальной предполагаемой цены сдачи объекта.
